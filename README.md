# KeyForge

KeyForge is a small desktop application for managing and storing entries in a local database.

The project is built with **Avalonia** and runs on **.NET 10**.  
Data is stored locally to keep the application simple and fast to use.

## Features

- Clean and structured desktop interface
- Local data storage
- Easily extendable
- Focus on usability

## Technologies

- C#
- Avalonia
- Entity Framework Core
- SQLite

## Project Goal

The goal of KeyForge is to provide a compact and modern application that allows data to be managed locally and reliably.
